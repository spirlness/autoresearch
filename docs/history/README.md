# History

This folder preserves historical material that is still useful for auditability and retrospectives, but is not part of the day-to-day operating path.

- [Benchmark history](./benchmarks/README.md) - committed raw benchmark records and older generated snapshots.
- [Conductor history](./conductor/README.md) - prior track plans, archives, and registry material.

For new local scratch output, prefer `results/` instead of adding more top-level historical folders.
